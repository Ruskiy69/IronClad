uint32_t ic::util::load_texture ( const char *  pfilename)

The caller is responsible for cleaning up the returned surface. If the image is found to have an alpha component, the return value comes from load_image_alpha instead. Pixel data is directly accessible like so: SDL_Surface* pSurf = ic::util::load_image("Test.png"); void* ppixels = pSurf->pixels;

Parameters:
char*Image path/filename
intFormat to load image as (optional=GL_RGB)
Returns:
An SDL_Surface* containing data about the image, or NULL if the image failed to load. Loads an image into GPU memory. This function uses the GLFW API to load the image located at the given filename. It is then loaded into GPU memory as a texture, and the texture handle is returned. If the image is found to have an alpha component, the returned texture handle is a product of load_texture_alpha instead. Only .tga files are supported!
Parameters:
char*Image filename/path.
intFormat to load the image into (optional=GL_RGB)
Returns:
The texture handle on successful loading, 0 (zero) on failure.
