Detailed Description

This class is a singleton, and should be globally accessible through ic::asset::g_AssetManager.

Only use the Find method if you are absolutely sure that the asset exists, otherwise use the Create method, which will create and load an asset if it does not exist or will return the existing one if it does.

This method will log an error if the asset fails to load.

The concept of someone "owning" an asset allows for multiple subsystems to load copies of it. For example, if you load a mesh into one scene, you tell the asset that &Scene1 is the owner. Anyone else that wants to use a reference to that asset must also provide the &Scene1 reference. If a different scene (say Scene2) wants to add the same mesh to its scene, it gives it the &Scene2 parameter, creating a new copy of the mesh rather than a reference to the one belonging to Scene1.
